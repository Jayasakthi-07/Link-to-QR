## ğŸ Getting Started

Run the project locally in 3 simple steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Jayasakthi-07/Link-to-QR.git
    cd Link-to-QR
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```
